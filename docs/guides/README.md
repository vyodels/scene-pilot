# docs/guides

This directory holds operator-facing or agent-facing usage guides.

Use it for:
- current tool usage contracts
- operator instructions
- agent execution guidance that is more concrete than specs but less temporary than a plan

Do not use it for:
- long-term canonical truth (`docs/specs/`)
- implementation tracking (`docs/plan/`)
- historical implementation notes (`docs/reference/`)
